# 📌 프로젝트 개요

본 프로젝트는 **학생(Student)**, **교수(Professor)**, **행정직원(Admin)**  
세 가지 역할을 기반으로 각각 다른 기능을 제공하는 **학사관리 시스템**입니다.

Python OOP + JSON 데이터 구조를 활용하여  
콘솔 환경에서 동작하는 **경량 학사관리 플랫폼**을 구현했습니다.

---

# 🧩 주요 기능

## 👨‍🎓 학생(Student)
- 수강신청 / 취소  
- 시간표 조회  
- 성적 조회  
- 공지사항 확인  
- 내 정보 수정  

## 👨‍🏫 교수(Professor)
- 담당 강의 조회  
- 수강생 명단 확인  
- 성적 입력  
- 공지사항 등록  

## 🧑‍💼 행정직원(Admin)
- 강의 개설 / 삭제  
- 전체 사용자 조회 및 관리
